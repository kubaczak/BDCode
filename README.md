# BDCode

App for sharing code through the internet.
## Installation

Use [yarn](https://yarnpkg.com/) to install dependencies.

```bash
cd server
yarn
```

And [npm](https://www.npmjs.com/) to install dependencies.

```bash
cd client
npm i
```
